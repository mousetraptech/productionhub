Audio — Avantis

/avantis

Allen & Heath Avantis digital mixer via MIDI TCP (port 51325)

Faders

AddressArgsDescription
/avantis/ch/{1-64}/mix/fader
 float 0.0-1.0 Input channel fader
/avantis/dca/{1-16}/fader
 float 0.0-1.0 DCA fader
/avantis/grp/{1-16}/mix/fader
 float 0.0-1.0 Group fader
/avantis/mix/{1-12}/mix/fader
 float 0.0-1.0 Mix/Aux fader
/avantis/fxsend/{1-4}/mix/fader
 float 0.0-1.0 FX Send fader
/avantis/fxrtn/{1-8}/mix/fader
 float 0.0-1.0 FX Return fader
/avantis/mtx/{1-6}/mix/fader
 float 0.0-1.0 Matrix fader
/avantis/main/mix/fader
 float 0.0-1.0 Main LR fader

Mutes

AddressArgsDescription
/avantis/ch/{1-64}/mix/mute
 int 1|0 Input channel mute
/avantis/dca/{1-16}/mute
 int 1|0 DCA mute
/avantis/main/mix/mute
 int 1|0 Main LR mute

Pan

AddressArgsDescription
/avantis/ch/{1-64}/mix/pan
 float 0.0-1.0 Pan (0=L, 0.5=C, 1=R)

Timed Fades

AddressArgsDescription
/avantis/ch/{n}/mix/fade
 float target, float secs, [string easing] Channel fader fade
/avantis/dca/{n}/fade
 float target, float secs, [string easing] DCA fade
/avantis/grp/{n}/mix/fade
 float target, float secs, [string easing] Group fade
/avantis/mix/{n}/mix/fade
 float target, float secs, [string easing] Mix/Aux fade
/avantis/main/mix/fade
 float target, float secs, [string easing] Main LR fade
Easing: linear, scurve (default), easein, easeout. Interpolated at 50Hz.

Scene

AddressArgsDescription
/avantis/scene/recall
 int 0-127 Recall scene by number

Fader Value Reference

ValueLevel
0.0-inf (silence)
0.5-12dB
0.75-6dB
1.00dB (unity)

Examples

Fade DCA 1 to silence over 3s: /avantis/dca/1/fade 0.0 3.0 scurve
Mute channel 5: /avantis/ch/5/mix/mute 1
Recall scene 3: /avantis/scene/recall 3

Lights — ChamSys QuickQ

/lights

ChamSys QuickQ lighting console via OSC relay (UDP port 8000)

Commands

AddressArgsDescription
/lights/pb/{N}/go
 int 1 Advance playback N to next cue
/lights/pb/{N}
 float 0.0-1.0 Set playback N fader level
/lights/pb/{N}/flash
 int 1|0 Flash playback (1=on, 0=off)
/lights/pb/{N}/pause
 (none) Pause playback N
/lights/pb/{N}/release
 (none) Release playback N
Use /pb/{N}/go 1 to advance to the next cue. Do NOT use /pb/{N}/1 — that jumps to cue number 1, not button 1.

Examples

Advance playback 1: /lights/pb/1/go 1
Set playback 2 to 50%: /lights/pb/2 0.5
Release playback 3: /lights/pb/3/release

Cameras — VISCA PTZ

/cam1 /cam2 /cam3

PTZ cameras via VISCA over IP. Each camera has its own prefix.

Commands

AddressArgsDescription
/cam{N}/preset/recall/{P}
 (none) Recall preset P (0-255)
/cam{N}/preset/store/{P}
 (none) Store current position as preset P
/cam{N}/home
 (none) Return to home position
/cam{N}/pantilt/speed
 float pan, float tilt Pan/tilt (-1.0 to 1.0 each)
/cam{N}/pantilt/stop
 (none) Stop pan/tilt movement
/cam{N}/pan/speed
 float -1.0 to 1.0 Pan only (neg=left, pos=right)
/cam{N}/tilt/speed
 float -1.0 to 1.0 Tilt only (neg=down, pos=up)
/cam{N}/zoom/speed
 float -1.0 to 1.0 Zoom (neg=wide, pos=tele)
/cam{N}/zoom/direct
 float 0.0-1.0 Set absolute zoom position
/cam{N}/zoom/stop
 (none) Stop zoom
/cam{N}/focus/auto
 (none) Enable auto focus
/cam{N}/focus/manual
 (none) Switch to manual focus
/cam{N}/power/on
 (none) Power on
/cam{N}/power/off
 (none) Power off

Examples

Recall preset 2 on camera 1: /cam1/preset/recall/2
Zoom camera 2 in at half speed: /cam2/zoom/speed 0.5
Stop camera 1 movement: /cam1/pantilt/stop

OBS Studio

/obs

OBS Studio via WebSocket v5 protocol (port 4455)

Commands

AddressArgsDescription
/obs/scene/{name}
 (none) Switch program scene
/obs/scene/preview/{name}
 (none) Switch preview scene
/obs/transition/trigger
 (none) Execute transition (preview to program)
/obs/transition/{name}
 (none) Set transition type
/obs/transition/duration
 int ms Set transition duration
/obs/stream/start
 (none) Start streaming
/obs/stream/stop
 (none) Stop streaming
/obs/stream/toggle
 (none) Toggle streaming
/obs/record/start
 (none) Start recording
/obs/record/stop
 (none) Stop recording
/obs/record/toggle
 (none) Toggle recording
/obs/virtualcam/start
 (none) Start virtual camera
/obs/virtualcam/stop
 (none) Stop virtual camera

Examples

Switch to Live scene: /obs/scene/Live
Start streaming: /obs/stream/start
Set 500ms fade transition: /obs/transition/Fade then /obs/transition/duration 500

TouchDesigner

/td

Transparent OSC relay to TouchDesigner (OSC In CHOP, port 12000)

Transparent OSC relay to TouchDesigner's OSC In CHOP. Any address is forwarded as-is (minus the /td prefix). Design your namespace in TD to match.

Example Addresses

AddressArgsDescription
/td/render/start
 (none) Trigger render
/td/render/stop
 (none) Stop render
/td/param/{name}
 float 0.0-1.0 Control named parameter
/td/cue/{n}
 int Trigger cue N
/td/opacity
 float 0.0-1.0 Master opacity
/td/blend/{layer}
 float 0.0-1.0 Layer blend level

Global / Hub Commands

System-level commands for cue sequencing, fade control, and hub status

Commands

AddressArgsDescription
/hub/go
 (none) Fire next cue in sequence
/hub/go/{cueId}
 (none) Fire specific cue by ID
/hub/stop
 (none) Stop/pause cue sequencer
/hub/back
 (none) Go back one cue
/hub/panic
 (none) Emergency: stop all fades + stop sequencer
/hub/status
 (none) Request status (replies via OSC)
/hub/cuelist/load
 string path Load cue list from file
/fade/stop
 (none) Stop all active fades
/fade/stop
 string key Stop specific fade by key

Examples

Fire next cue: /hub/go
Emergency stop: /hub/panic
Stop all fades: /fade/stop
